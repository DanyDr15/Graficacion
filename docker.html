INSTALACION DE DOCKER

La idea detrás de Docker es crear contenedores ligeros y portables para las aplicaciones software que puedan ejecutarse en cualquier máquina con Docker instalado, independientemente del sistema operativo que la máquina tenga por debajo, facilitando así también los despliegues. Docker, me permite meter en un contenedor (“una caja”, algo auto contenido, cerrado) todas aquellas cosas que mi aplicación necesita para ser ejecutada (java, Maven, tomcat…) y la propia aplicación. Así yo me puedo llevar ese contenedor a cualquier máquina que tenga instalado Docker y ejecutar la aplicación sin tener que hacer nada más. INSTALACION DEL SERVIDOR DIGITAL OCEAN

Para empezar necesitamos crear una cuenta en Digital Ocean, es la pagina donde compraremos nuestro servidor. Existen varios precios dependiendo en que lo vamos a usar, como el de nosotros es para la universidad contratar el que cuesta 5$ es suficiente. Luego empezamos a crear droplet que vendria siendo nuestro sistema y nuestro servidor. En el por medio de la consola instalaremos docker como hemos visto anteriormente y continuamos instalando el servidor apache con el siguiente comando docker run -p 80:80 --name web -d -v /htdocs: /usr/local/apache2/htdocs httpd y listo ya podemos ir poniendo todo lo que queramos hacer en la carpeta htdocs.
